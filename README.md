# Career Tree
就職活動における企業管理・締切管理を効率化するために開発した、Streamlit製の就活管理システムです。
夏季インターンシップへの応募企業が増える中で、Excelによる管理では締切や合否通知日の見落としが発生していたため、自身の課題解決を目的として開発しました。

## 開発環境
* Python
* VS Code
* Streamlit
* Git / GitHub
* Pandas

## 主な機能
### 企業管理
* 企業情報の登録・修正・削除
* 企業ID管理
* 企業ホームページ管理
### 選考管理
* プレエントリー
* ES提出
* 適性検査
* 面接
* 合否通知
* 内定・落選管理
### スケジュール管理
* 締切アラート
* インターン開始アラート
* 日程管理
### 検索・分析
* 企業検索
* ステータス絞り込み
* 並び替え
* ダッシュボード表示
